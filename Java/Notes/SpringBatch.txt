SPRING BATCH

TASKLET
  É utilizado para uma tarefa única (fazer um calculo, acessar uma URL, fazer uma tarefa de um API)
  
CHUNK:
  Processa lotes de dados, fila de mensageria e etc;
  
  
Spring Batch - visão geral:
  - JobLauncher pode ser uma API q dispare um Job que é formado por vários steps
    - As steps são divididas em:
      - Reader (leitura da mensagem)
      - Processor (filtro)
      - Writer (grava um arquivo, ou um banco de dados);
      
    - Os jobs permitem passar parâmetros e realizar tentativas
    
Princípios e orientações para uma aplicação Batch:
  - O Batch deve rodar de madrugada;
  - Simplificar o máximo o job (não colocar tudo em um projeto);
  - Conferir o header (quantidade de processamento);
  - Fazer testes com uma gama de registros grandes (3 milhões)
  - Backup do arquivo q esta sendo processado;
  - O processo tem q ser capaz de reprocessar o arquivo caso não consiga.
